# What to do next

1. Push this repo to `git@github.com:adityasomde/printease-desk.git`.
2. Run the local test commands from `README.md`.
3. Push tag `desktop-v0.1.1`.
4. Download the `.AppImage` from GitHub Releases.
5. Run that AppImage on Linux. Future tagged releases should be detected by the updater.

Important: do not commit `.env`, backend code, database URLs, Supabase service keys, JWT secrets, or `node_modules`.
